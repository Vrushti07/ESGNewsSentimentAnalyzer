# Notes
# ─────────────────────────────────────────────────────────────────────────────
# 1. First run downloads ~500 MB of model weights (FinBERT + MiniLM).
#    They are cached in ~/.cache/huggingface/ by default.
#
# 2. On Apple Silicon (M1/M2), PyTorch works natively — no CUDA needed.
#    Install via:  pip install torch  (picks the right wheel automatically)
#
# 3. SHAP's text masker requires the tokenizer from the FinBERT pipeline.
#    All three are wired together inside app.py via @st.cache_resource.
#
# 4. scipy is a transitive dep of shap — no explicit pin needed unless you
#    hit conflicts; add  scipy>=1.12.0  in that case.
#
# 5. To run:
#       streamlit run app.py
